    Class BasePipeline<TExec, TResult>

    Converts extracted metadata into downloadable work items.

    Remarks

    Pipelines exist because extraction output is descriptive, while downloads need concrete URLs, media types, file extensions, and stable identifiers. Provider pipelines decide which media are eligible and how files should be grouped on disk.
